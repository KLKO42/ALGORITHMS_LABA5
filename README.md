## Задача 1 — Бинарное дерево поиска (BST)

### Описание

Реализовать **бинарное дерево поиска** для хранения целых чисел. Каждый узел содержит значение, а также ссылки на левого и правого потомка. Для любого узла все элементы в левом поддереве меньше его значения, а в правом - больше.

---

### Обязательная часть

Реализовать функции:
- `insert(root, value)`: вставить значение в дерево поиска. При необходимости создавать корневой узел.(есть)
- `search(root, target)`: найти, содержится ли значение в дереве (возвращать True/False или сам узел).(есть)
- `inorder(root)`: вывести значения в порядке возрастания (симметричный обход).(есть)
